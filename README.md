# Categorical Data Preprocessing in Machine Learning

## Overview
This project demonstrates different categorical data preprocessing techniques using Python and Scikit-learn.

The notebook explains how to convert categorical features into numerical values before training machine learning models.

## Techniques Implemented

- One-Hot Encoding
- Label Encoding
- Ordinal Encoding
- Column Transformer
- Train-Test Split
- Feature and Target Separation

## Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Workflow

1. Load Dataset
2. Explore Categorical Columns
3. Split Features and Target
4. Train-Test Split
5. Apply One-Hot Encoding
6. Apply Label Encoding
7. Apply Multiple Preprocessing Methods using ColumnTransformer
8. Generate Processed Dataset
